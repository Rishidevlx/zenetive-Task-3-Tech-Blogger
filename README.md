# 🚀 Tech Blogger

A modern **Tech Blogging Platform** where users can create, publish, and explore technology-related blogs.
This project demonstrates the implementation of a **full-stack blogging system** with structured backend APIs and a responsive frontend.

---

## 🌐 Project Overview

Tech Blogger is a blogging application designed for sharing knowledge related to technology.
Users can create blog posts, view articles written by others, and manage their own content efficiently.

The project focuses on implementing **core blogging functionalities**, including post creation, content management, and structured server-side architecture.

---

## ✨ Features

* User Authentication System
* Create Blog Posts
* Edit Existing Blogs
* Delete Blog Posts
* View Blogs from Other Users
* Responsive User Interface
* Organized Backend API Structure
* Full CRUD Blog Operations

## ⚙️ Installation & Setup

Clone the repository

```
git clone https://github.com/Rishidevlx/zenetive-Task-3-Tech-Blogger.git
```

Navigate to the project directory

```
cd zenetive-Task-3-Tech-Blogger
```

Install dependencies

```
npm install
```

Run the application

```
npm start
```
